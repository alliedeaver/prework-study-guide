HTML

  • The HTML is used to create the structure of a webpage, using elements to provide content
  • The body element represents the visible content shown to the user
  • The child elements are elements inside other elements
  • The element that holds all the children elements togther, or the body, is the parent element
  • HMTL is one of the three cornerstone languages used in web developement
  • The head element contains information about the webpage

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is a programming language used by web developers to define how the webpage content will be presented to the users
  • Makes up the aesthetics, color, visual layout and organization
  • A single change to the CSS rule will apply to all tagged elements
  • The most common way to style a webpage is using the external CSS
  • In the CSS style sheet, keep all CSS rules in a separate file, making design changes easier
  • A margin defines how much space is wanted around the outside of the element
  • A padding defines how much space is wanted arounf the insdie of the element

Git

  • git status: checks what branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • The Git command must precede every statement in the CLI
  • git add means adding modifications in the current working branch
  • git add -A indicates we want to add all changes
  • git pull origin command to always have the latest version
  • git commit: save changes, makes a commit

Javascript

  • Javascript is what makes the application interactive
  • Javascript is versatile and easy to use but also dynamic
  • Javascript is a high level programming language which at runtume will execute the programming behaviors
  • The variables in Javascript are not directly associated with any particular value type, and any variable can be assigned/re-assigned
  • Undefined indicates the asbence of a value, and is also a normal identifier
  • Null indicates the absence of an object and is a keyword
  • Javascript uses control flow, which is how the computer will execute the code and does so top to bottom
  • Conditional statements and loops will control what order the code is executed. That is if a condition is met, or found as truthy, then the command will be executed. If the condition is false, or falsy, then the code will not be executed
